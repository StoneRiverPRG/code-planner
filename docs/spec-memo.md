# spec メモ

## ユーザー記入

### やりたいこと

- 3×3の○×ゲームをCUIで
- AIと人間（ユーザー）が対戦する
- AIはランダムで手を指す

### 断片的な要素

- 先攻後攻はランダム
- CUIで○×の現在の状態を表示する

## AI記入（以下にAIが整形した仕様メモを追記する）

### 確定事項 (2026-03-18)
- **入力形式**: 1〜9の数値入力（グリッドを左上から右下に1-9で対応）。
- **エラーハンドリング**: 範囲外、既埋め、数以外に対してエラーメッセージを表示し、再入力を促す。
- **終了条件**: 勝敗確定または引き分け時に結果(Winner/Draw)を表示して即時終了。
- **プロジェクト名**: `tic-tac-toe`
